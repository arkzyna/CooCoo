Detailed Description

Classe modélisant les complexes. Elle hérite de Donnee.

Author:
Letellier/Yassine

Pour fonctionner cette classe encapsule deux Constantes : sa partie réelle et sa partie imaginaire. Tous les opérateurs sont égalements redéfinis.
