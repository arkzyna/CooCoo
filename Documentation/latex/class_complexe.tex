\subsection{Detailed Description}
Classe modélisant les complexes. Elle hérite de \hyperlink{class_donnee}{Donnee}. 

\begin{DoxyAuthor}{Author}
Letellier/\-Yassine
\end{DoxyAuthor}
Pour fonctionner cette classe encapsule deux Constantes \-: sa partie réelle et sa partie imaginaire. Tous les opérateurs sont égalements redéfinis.
